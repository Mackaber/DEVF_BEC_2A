# Aprende a montar un servidor en Node.js usando Express.

- Configurar un servidor Express básico.
- Crear rutas GET y POST.
- Enviar respuestas en formato JSON.
- Probar endpoints localmente.

```bash
# Paso 1. Comienza un proyecto de forma interactiva para Node.js
npm init

# Paso 2. Instalar Express JS
npm install express

# Paso 7. Correr el archivo
node index.js
npm start # Si es que ya existe "scripts": { "start": node index.js } en package.json
```
